# gruppo1se

Per implementare alcune funzionalità del sistema è stata utilizzata una libreria esterna, scaricabile al seguente link: https://www.toedter.com/download/jcalendar-1.4.zip 

È necessario importare tra le librerie all'interno del progetto il file jcalendar-1.4.jar che si trova nella cartella lib all'interno dell'archivio jcalendar-1.4.zip.
